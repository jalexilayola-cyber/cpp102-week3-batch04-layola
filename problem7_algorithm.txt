PROBLEM 7 ALGORITHM

Step 2: Ask the user to enter their lab work score. Store it as lab work.
Step 3: Ask the user to enter their attendace. Store it as attendace.
Step 4: Ask the user to enter their quiz scores. Store it as quiz score.
Step 5: Compute laboratory standing = (lab work * 40%) +
